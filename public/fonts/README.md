# Font Files

## Required Font
Place the Household Words font file here:
- File name: `HouseholdWords.ttf` or `HouseholdWords.otf`
- Download from: https://www.dafont.com/householdwords.font

## Installation
1. Download the font from DaFont
2. Extract the .ttf or .otf file
3. Place it in this directory as `HouseholdWords.ttf`
4. Run `npm run build` from the project root
5. Clear browser cache
