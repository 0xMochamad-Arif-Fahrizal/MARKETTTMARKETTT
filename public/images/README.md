# Logo Image

## Required File
Place your logo image here:
- File name: `logo.png` (or `logo.svg` for vector)
- Recommended size: Height 32-40px (width auto)
- Format: PNG with transparent background or SVG

## Installation
1. Prepare your logo file
2. Save it as `logo.png` in this directory
3. If using SVG, update AppLayout.vue to use `logo.svg` instead
4. Refresh browser to see the logo
